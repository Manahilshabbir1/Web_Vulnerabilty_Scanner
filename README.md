# Web_Vulnerability_Scanner
Python + Flask based scanner for detecting common web vulnerabilities.

##  Features
- Security Header Analysis
- SQL Injection Detection (basic)
- XSS Detection (basic)
- Sensitive File Exposure Check
- Form Detection
- Scan History Storage

## Tech Stack
Python, Flask, Requests, BeautifulSoup

##  Safe Testing Targets
- OWASP Juice Shop
- OWASP WebGoat
